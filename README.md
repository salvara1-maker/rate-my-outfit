# Rate My Outfit

## Overview

Rate My Outfit is a mobile app where users can upload pictures of their outfits and receive ratings. The app helps users get quick feedback and feel more confident before going out.

---

## Goals

* Provide quick outfit feedback
* Create a fun and social experience
* Encourage daily engagement

---

## App Evaluation Criteria

### Mobile

* Uses camera for uploading outfits
* Designed for quick mobile interactions
* Not a static website experience

### Story

Users often want quick feedback before going out. This app solves that problem by providing instant ratings.

### Market

* Target audience: students and young adults
* Large user base due to social + fashion appeal

### Habit

* Users can post daily outfits
* Check ratings regularly
* Engage with others

### Scope

* MVP is simple and achievable
* Core features are clearly defined
* Expandable with social features
